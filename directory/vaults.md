# Starter Vaults & Templates Directory

> Public starter vaults, template systems, and reference vaults you can browse, fork, or learn from.
>
> Last reviewed: 2026-04-25.

## Starter Vaults

- [**Ideaverse for Obsidian (LYT)**](https://www.linkingyourthinking.com) — Nick Milo's flagship starter vault and template system, built around the LYT framework and Maps of Content.
- [**Obsidian Sample Vault**](https://github.com/obsidianmd/obsidian-sample-plugin) — Official sample vault structure included with the plugin development template.

## Curated Hubs

- [**Obsidian Hub**](https://publish.obsidian.md/hub) — Browseable community-curated vault published with Obsidian Publish. Contains resources, talks, templates, and showcases. The vault itself can be downloaded from [obsidian-community/obsidian-hub](https://github.com/obsidian-community/obsidian-hub).

## Categories We're Tracking

The following categories of public vaults exist but require per-vault verification before listing:

- **Personal published vaults** — Vaults the author publishes via Obsidian Publish or Quartz as a digital garden.
- **Topic-specific reference vaults** — Vaults built around a single domain (history, philosophy, programming, TTRPG worldbuilding, etc.).
- **Educational starter vaults** — Vaults designed for students or specific professions (academic, legal, medical, engineering).

## Submit a Vault

If you maintain a public starter vault, template pack, or reference vault that other Obsidian users could fork or learn from, open a PR. See [CONTRIBUTING.md](../CONTRIBUTING.md). The vault must be publicly accessible (GitHub repo or Obsidian Publish URL) and actively maintained.
