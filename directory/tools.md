# Tools & Integrations Directory

> External tools, services, and standalone software that pair well with Obsidian. Plugins live in [plugins.md](plugins.md); this file is for things outside the vault.
>
> Last reviewed: 2026-04-25.

## Publishing & Sites from Vaults

- [**Quartz**](https://github.com/jackyzha0/quartz) — Free, open-source static-site generator that turns an Obsidian vault into a public digital garden. Wikilinks, backlinks, graph view, full-text search. Docs at [quartz.jzhao.xyz](https://quartz.jzhao.xyz). Built by Jacky Zhao.
- [**Obsidian Publish**](https://obsidian.md/publish) — Official paid hosted publishing service from the Obsidian team. Best for users who want zero-config publishing with native feature parity.
- [**Hugo**](https://gohugo.io) — General-purpose static-site generator with Obsidian-flavored Markdown via shortcodes and plugins.

## Web Clippers & Capture

- [**Obsidian Web Clipper**](https://obsidian.md/clipper) — Official browser extension. Free, open source, fully local. Captures web content directly into the vault.
- [**Readwise Reader**](https://readwise.io/read) — Read-later app with automatic highlight sync into Obsidian via the Readwise plugin.

## Sync & Backup (Standalone)

- [**Syncthing**](https://syncthing.net) — Self-hosted peer-to-peer file sync with no third-party servers. Most popular non-cloud option for vault sync.
- [**Obsidian Sync**](https://obsidian.md/sync) — Official paid end-to-end-encrypted sync from the Obsidian team.

## Reference & Citation Management

- [**Zotero**](https://www.zotero.org) — Open-source reference manager. Pairs with the Citations and Zotero Integration plugins for academic workflows.

## Document Conversion & Export

- [**Pandoc**](https://pandoc.org) — Universal document converter (the standalone binary). Required by the Pandoc and Enhancing Export plugins for DOCX, ePub, PDF, and other formats.

## Audio, Video, Transcription

- [**yt-dlp**](https://github.com/yt-dlp/yt-dlp) — Command-line video/audio downloader. Useful for piping YouTube content into the vault for transcription or notes.
- [**Whisper.cpp**](https://github.com/ggerganov/whisper.cpp) — Local C++ port of OpenAI Whisper for on-device speech-to-text. Pairs with the Transcription plugin for fully local STT.

## Local AI / LLM Backends

- [**Ollama**](https://ollama.com) — Run open-weight LLMs locally with a one-line install. Used by Smart Connections, Copilot, Text Generator, and most local-first AI plugins.
- [**LM Studio**](https://lmstudio.ai) — Desktop app for running local LLMs with an OpenAI-compatible API. Alternative to Ollama with a GUI.

## Scraping & Data Pipelines

- [**Apify Platform**](https://apify.com) — Custom web scrapers for feeding external data into the vault.
