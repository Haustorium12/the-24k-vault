# Communities Directory

> Official and community-run forums, Discord servers, subreddits, and meeting places for Obsidian users.
>
> Last reviewed: 2026-04-25.

## Official

- [**Obsidian Forum**](https://forum.obsidian.md) — Official forum: methodology discussion, plugin development, help, feature requests. The most active long-form Obsidian community.
- [**Obsidian Discord**](https://discord.com/invite/obby) — Official community Discord. Real-time help, plugin dev chat, ecosystem news. Invite code `obby`.
- [**Obsidian Community Page**](https://obsidian.md/community) — Aggregator with links to Discord, forum, roadmap, and other official channels.

## Community-Run

- [**r/ObsidianMD**](https://www.reddit.com/r/ObsidianMD) — Subreddit. Workflows, screenshots, plugin discussion, community support.
- [**Obsidian Hub**](https://publish.obsidian.md/hub) — Community-curated wiki of resources, plugins, themes, vaults, talks, and templates. Maintained by [obsidian-community/obsidian-hub](https://github.com/obsidian-community/obsidian-hub).
- [**Vault Hub**](https://obsidianvaulthub.com) — Independent discovery site for Obsidian plugins and resources.

## Niche / Specialized

- [**Obsidian-TTRPG-Community**](https://github.com/Obsidian-TTRPG-Community) — GitHub organization maintaining Fantasy Statblocks, Dice Roller, Initiative Tracker, and other tabletop RPG plugins. The hub for TTRPG users on Obsidian.
